# ✅ Todo App - React 

## 📌 Overview
A simple and efficient Todo application built with React that allows users to manage daily tasks with CRUD operations and a clean user-friendly interface.

---

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- JavaScript (ES6+)

---

## ✨ Features
- ➕ Add tasks
- ✏️ Edit tasks
- ❌ Delete tasks
- ✅ Mark as complete
- 🔍 Filter tasks
- 📱 Responsive UI

---

## 🧠 Key Learnings
- React state management
- CRUD operations
- Component-based architecture

---

git clone https://github.com/yuvraj0651/Todo-App-React.git
